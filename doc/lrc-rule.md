# lrc 文件格式

```txt
1.标识标签 [name:item]

[ar:歌手名]
[ti:歌曲名]
[al:专辑名]
[by:lrc制作人]
[au:作词者]
[re:编辑软件]
[ve:版本]
[offest:时间补偿值]

2.时间标识 [mm:ss.ms]

标准[分钟:秒.毫秒]
其它[分钟.秒]
其它[分钟:秒:秒]

3.示例

[ti:Heart To Heart]
[ar:James Blunt]
[al:Heart To Heart]
[by:]
[offset:0]
[00:00.00]Heart To Heart (诚恳的) - James Blunt (詹姆仕·布朗特)
...
```
